# Documentación - Capturas y Guías

Esta carpeta contiene documentación visual, capturas de pantalla y guías del proyecto.

## 📁 Estructura

```
docs/
├── README.md                    # Este archivo
├── screenshots/                 # Capturas de pantalla de la aplicación
├── api-documentation.md         # [PRÓXIMO] Documentación de API endpoints
└── user-guide.md               # [PRÓXIMO] Guía de usuario
```

## 📸 Screenshots

Se agregará capturas de:
- Dashboard principal
- Módulos de matrícula, cursos, notas, etc.
- Interfaz de administración

## 📖 Documentación

Se incluirá:
- Guía de uso de cada módulo
- Documentación completa de API
- Diagramas de flujo
- Guías de instalación (ver README.md raíz)

## 🔄 Cómo Contribuir Docs

1. Agregar capturas en `screenshots/`
2. Actualizar `api-documentation.md` con nuevos endpoints
3. Documentar nuevos módulos en `user-guide.md`
